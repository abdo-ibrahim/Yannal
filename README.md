# Yannal

Yannal is a responsive landing page built with **HTML, CSS, and JavaScript**. It supports **English and Arabic languages** with dynamic translation, making it easy to switch languages on the fly.  

## Demo
[https://abdo-ibrahim.github.io/Yannal](https://abdo-ibrahim.github.io/Yannal/)

## Features

- Fully responsive design (mobile, tablet, desktop)
- Multi-language support (English & Arabic)
- Dynamic text translation using JSON or i18next
- RTL support for Arabic
- Clean and modern UI
